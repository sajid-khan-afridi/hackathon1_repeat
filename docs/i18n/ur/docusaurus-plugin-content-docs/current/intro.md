# فزیکل AI اور ہیومنائیڈ روبوٹکس میں خوش آمدید

یہ نصابی کتاب آپ کو جدید روبوٹکس کی بنیادی باتوں سے لے کر پیشرفتہ تصورات تک لے جائے گی۔

## آپ کیا سیکھیں گے

اس نصابی کتاب میں، آپ یہ سیکھیں گے:

- **ROS 2 بنیادیات**: روبوٹ آپریٹنگ سسٹم کی جدید نسل
  - Nodes اور communication patterns
  - Topics، Services، اور Actions
  - Launch files اور Parameters

- **NVIDIA Isaac Sim**: فزکس بیسڈ روبوٹ سیمولیشن
  - URDF روبوٹ ماڈلز
  - Sensors اور perception
  - Isaac ROS انضمام

- **ہیومنائیڈ روبوٹکس**: انسانی شکل کے روبوٹس
  - Bipedal locomotion
  - Kinematics اور dynamics
  - Motion planning

## پیش نظارہ کیسے استعمال کریں

:::tip نیویگیشن
- بائیں سائڈبار میں ماڈیولز دیکھیں
- ہر باب میں کوڈ مثالیں شامل ہیں
- تکنیکی اصطلاحات انگریزی میں محفوظ ہیں
:::

## تکنیکی اصطلاحات

اس نصابی کتاب میں، تکنیکی اصطلاحات جیسے <GlossaryTooltip termId="ros-2">ROS 2</GlossaryTooltip>, <GlossaryTooltip termId="python">Python</GlossaryTooltip>, <GlossaryTooltip termId="urdf">URDF</GlossaryTooltip>, اور **Isaac Sim** انگریزی میں رکھی گئی ہیں تاکہ آپ آسانی سے دستاویزات اور کوڈ کے ساتھ کام کر سکیں۔

:::info اصطلاحات کی وضاحت
تکنیکی اصطلاحات پر ہوور کریں تاکہ اردو اور انگریزی میں تعریف دیکھ سکیں۔ مکمل اصطلاحات کی فہرست کے لیے [اصطلاحات کا صفحہ](/glossary) دیکھیں۔
:::
