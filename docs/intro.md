# Welcome to the Physical AI & Humanoid Robotics Textbook

This comprehensive textbook covers the essential concepts and practical skills needed to work with modern robotics systems, from basic ROS 2 programming to advanced AI integration with humanoid robots.

## What You'll Learn

This textbook is organized into modules that build upon each other:

- **Module 1: ROS 2 Fundamentals** - Master the Robot Operating System
- **Module 2: NVIDIA Isaac Sim** - Learn physics-based simulation
- **Module 3: Computer Vision** - Visual perception for robots
- **Module 4: Motion Planning** - Path planning and control
- **Module 5: Machine Learning for Robotics** - AI-powered robot behaviors

## Getting Started

Each module includes hands-on examples, code samples, and practical exercises. All code is provided in <GlossaryTooltip termId="python">Python</GlossaryTooltip> and C++ to accommodate different programming backgrounds.

### Prerequisites

- Basic programming knowledge (Python or C++)
- Linux command line familiarity
- Understanding of basic robotics concepts

## Course Structure

- **Chapters**: Each module contains multiple chapters focusing on specific topics
- **Code Examples**: Practical, runnable code for every concept
- **Exercises**: Hands-on activities to reinforce learning
- **Projects**: Comprehensive projects that combine multiple concepts
